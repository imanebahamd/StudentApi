# Student API

A **RESTful API built with .NET 8** to manage students with full **CRUD operations**.  

The project follows **Domain-Driven Design (DDD)** and implements **CQRS** using the **MediatR** package. It uses **SQL Server** as the database and **Entity Framework Core** for data access. Swagger UI is included for testing the endpoints.

---

## Features

- **Get all students**: `GET /api/students`  
- **Get a student by ID**: `GET /api/students/{id}`  
- **Create a new student**: `POST /api/students`  
- **Update a student**: `PUT /api/students/{id}`  
- **Delete a student**: `DELETE /api/students/{id}`
